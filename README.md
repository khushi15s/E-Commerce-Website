# 🛒E-Commerce Web Page

A modern and user-friendly **E-Commerce Web Application** built using **HTML, CSS, and JavaScript**.
This project features a stylish UI inspired by modern shopping websites and includes essential shopping functionalities like search, cart, checkout, and product ratings.

---

## 🌟 Features

* 🎨 Attractive **gradient UI (purple/pink theme)**
* 🧭 Easy-to-use **navigation bar**
* 🔍 **Search functionality** for products
* 🛍 Add products to **shopping cart**
* 💰 **Total amount calculation**
* 🧾 Simple **checkout system**
* ⭐ Product **ratings display**
* 💬 **Customer support section**
* 💻 Optimized for **laptop/desktop view**

---

## 📁 Project Structure
ecommerce-website/
│
├── index.html     # Main structure of the webpage
├── style.css      # Styling and layout design
└── script.js      # Functionality (cart, search, checkout)
```

---

## 🚀 How to Run the Project

1. Download or copy all the files:

   * `index.html`
   * `style.css`
   * `script.js`

2. Place all files in the **same folder**

3. Open `index.html` in any web browser

## 🧩 Functional Modules

### 🔍 Search System

* Filters products in real-time based on user input

### 🛒 Shopping Cart

* Add items to cart
* Displays selected items
* Updates total price dynamically

### 💰 Checkout

* Simple checkout button to simulate order placement
* Clears cart after checkout

### ⭐ Product Ratings

* Displays ratings using star symbols

### 💬 Customer Support

* Includes contact information for users
